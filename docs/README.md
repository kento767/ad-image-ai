## 開発の進め方

1. [requirements.md](./requirements.md) で要件を確認
2. [TASKS.md](./TASKS.md) から着手するタスクを選ぶ
3. [DEVELOPMENT.md](./DEVELOPMENT.md) に従って環境をセットアップ
4. [ARCHITECTURE.md](./ARCHITECTURE.md) を参照しながら実装
5. 完了したら [TASKS.md](./TASKS.md) を更新し、[CHANGELOG.md](./CHANGELOG.md) に記録
